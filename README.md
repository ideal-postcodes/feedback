# Ideal Postcodes — Community Feedback

Public issue tracker for [Ideal Postcodes](https://ideal-postcodes.co.uk) products and tools.

Use it to:

- Report a bug
- Request a feature or change
- Ask a question that isn't account- or billing-specific

## What this tracker covers

- **`@ideal-postcodes/cli`** — the official `idpc` command-line tool
- **Ideal Postcodes API** — the HTTP API at `api.ideal-postcodes.co.uk`
- **Address Finder Library** (`@ideal-postcodes/address-finder`) and **Postcode Lookup Library** (`@ideal-postcodes/postcode-lookup`)
- **Agent skills** — packages under [`ideal-postcodes/skills`](https://github.com/ideal-postcodes/skills)
- **Documentation** at [docs.ideal-postcodes.co.uk](https://docs.ideal-postcodes.co.uk)
- **General questions** about products, integration patterns, or pricing direction

## What this tracker does *not* cover

- **Account, billing, or sensitive issues** → email [support@ideal-postcodes.co.uk](mailto:support@ideal-postcodes.co.uk).
  **Do not post API keys, user tokens, addresses, or other personal data here.** Issues are public.
- **Security vulnerabilities** → see [SECURITY.md](./SECURITY.md). Do not file them publicly.
- **Ecommerce integrations** — these have their own public repos:
  - [`woocommerce`](https://github.com/ideal-postcodes/woocommerce) · [`magento`](https://github.com/ideal-postcodes/magento) · [`shopify`](https://github.com/ideal-postcodes/shopify) · [`bigcommerce-app`](https://github.com/ideal-postcodes/bigcommerce-app) · [`salesforce`](https://github.com/ideal-postcodes/salesforce)

## Filing an issue

**Humans:** open the [Issues tab](https://github.com/ideal-postcodes/feedback/issues/new/choose) and pick a template. The first field is a **product** dropdown — please pick the right one so the issue is routed correctly.

**Agents and CLI users:** see [AGENTS.md](./AGENTS.md) for the `gh` / API contract. There are markdown templates under `.github/ISSUE_TEMPLATE/` (`bug_report.md`, `feature_request.md`) you can use as a body template.

## Documentation

See [docs.ideal-postcodes.co.uk](https://docs.ideal-postcodes.co.uk) before filing — many "is this a bug?" questions are answered there.
